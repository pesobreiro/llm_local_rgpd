## CLÁUSULA 3 - OBRIGAÇÕES DO SUBCONTRATANTE

### 3.1 Instruções Documentadas (Art. 28º, nº 3, a))

O Subcontratante compromete-se a:
- [ ] Tratar dados pessoais **apenas mediante instruções documentadas** do Responsável pelo Tratamento
- [ ] Não utilizar dados para fins diferentes dos acordados
- [ ] Não divulgar dados a terceiros sem autorização prévia e por escrito
- [ ] Informar imediatamente o Responsável se uma instrução violar o RGPD

### 3.2 Confidencialidade (Art. 28º, nº 3, b))

O Subcontratante garante que:
- [ ] Pessoas autorizadas assumiram compromisso de confidencialidade
- [ ] Existência de obrigações legais de confidencialidade aplicáveis
- [ ] Medidas de controlo de acesso implementadas

### 3.3 Medidas de Segurança (Art. 28º, nº 3, c) e Art. 32º)

O Subcontratante implementa as seguintes medidas:

#### Medidas Técnicas:
- [ ] Encriptação de dados em trânsito (TLS 1.3)
- [ ] Encriptação de dados em repouso (AES-256)
- [ ] Pseudonimização quando aplicável
- [ ] Controlo de acesso baseado em funções (RBAC)
- [ ] Autenticação multifator (MFA)
- [ ] Registo de logs de auditoria

#### Medidas Organizacionais:
- [ ] Política de segurança da informação
- [ ] Procedimentos de resposta a incidentes
- [ ] Formação de pessoal em proteção de dados
- [ ] Avaliação periódica de riscos

### 3.4 Subcontratantes Ulteriores (Art. 28º, nº 2 e 4))

- [ ] **Não autorizado:** O Subcontratante não pode contratar outros subcontratantes sem autorização prévia e por escrito do Responsável pelo Tratamento
- [ ] **Autorização geral:** [LISTAR SUBCONTRATANTES AUTORIZADOS, SE APLICÁVEL]
- [ ] **Obrigações:** Subcontratantes ulteriores ficam vinculados às mesmas obrigações do presente contrato

### 3.5 Assistência ao Responsável (Art. 28º, nº 3, e) e f))

O Subcontratante prestará assistência para:
- [ ] Responder a pedidos dos titulares dos dados (acesso, retificação, apagamento, etc.)
- [ ] Garantir cumprimento das obrigações do Art. 32º a 36º do RGPD
- [ ] Fornecer informações necessárias para demonstrar conformidade
- [ ] Permitir e contribuir para auditorias/inspeções

### 3.6 Devolução/Eliminação de Dados (Art. 28º, nº 3, g))

Após conclusão do tratamento:
- [ ] **Opção 1:** Devolver todos os dados pessoais ao Responsável
- [ ] **Opção 2:** Eliminar todos os dados pessoais e cópias existentes
- [ ] **Certificação:** Fornecer comprovação de eliminação

**Exceção:** Dados que devam ser conservados por obrigação legal

### 4.1 Fornecimento de Instruções

O Responsável pelo Tratamento compromete-se a:
- Fornecer instruções claras e documentadas ao Subcontratante
- Informar sobre quaisquer alterações às instruções
- Responder a questões de interpretação no prazo de [NÚMERO] dias

### 4.2 Supervisão

O Responsável pelo Tratamento tem o direito de:
- Auditar o Subcontratante periodicamente
- Receber relatórios de conformidade
- Inspecionar instalações (mediante acordo prévio)

### 4.3 Informação aos Titulares

O Responsável pelo Tratamento mantém a obrigação de:
- Informar titulares sobre a existência do subcontratante
- Indicar categorias de destinatários (incluindo subcontratantes)

## CLÁUSULA 6 - INCIDENTES DE SEGURANÇA

### 6.1 Notificação de Violações

O Subcontratante notificará o Responsável pelo Tratamento:
- **Imediatamente** após tomar conhecimento de uma violação de dados pessoais
- No máximo, dentro de **24 horas** após deteção

### 6.2 Conteúdo da Notificação

A notificação inicial deve incluir:
- [ ] Descrição da violação (natureza, categorias de dados, número aproximado de titulares)
- [ ] Prováveis consequências
- [ ] Medidas já tomadas ou propostas
- [ ] Contacto para obtenção de mais informações

### 6.3 Documentação

O Subcontratante documenta:
- Todos os factos relativos à violação
- Efeitos da violação
- Medidas corretivas tomadas
